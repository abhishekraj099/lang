# Needed From You Later

No API keys are needed for the current local-first course phase.

## AI Tutor Phase
- Gemini API key.
- Preferred tutor names and personalities.
- Daily free AI conversation limit.
- Whether explanations should be English, Hindi, or Hinglish.

## Supabase Sync Phase
- Supabase project URL.
- Supabase anon public key.
- Google OAuth client details for Android.
- Android SHA-1 fingerprint added in Google Cloud Console.
- Whether users must be allowed to try lessons before login.

## Firebase Analytics And Crash Reporting Phase
- Firebase project config file (`google-services.json`).
- Confirmation that analytics/crash reporting is okay for your privacy policy.

## Monetization Phase
- Google Play developer account access.
- AdMob app ID and ad unit IDs.
- Play Billing product IDs and pricing.
- Whether the app should show ads from day one or only after beta testing.

## Branding And Store Phase
- Final app name.
- App icon or preferred style direction.
- Short description, full description, and screenshot preferences.
- Privacy policy URL.
- Support email address for Play Store listing.
